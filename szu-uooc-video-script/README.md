# UOOC 优课自动刷课脚本

适用于深圳大学（SZTU）软件工程等课程的 UOOC 平台网课自动播放脚本。

## 功能

- 自动遍历所有章节、节，查找未完成的视频
- 自动播放视频（2 倍速、静音）
- 自动跳过章节测试/测验
- 播完后刷新页面，继续检测是否有遗漏

## 使用方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开 `szu慕课网课刷课脚本.js`，复制全部代码
3. 点击 Tampermonkey 图标 → **管理面板** → **添加新脚本**
4. 粘贴代码，按 `Ctrl+S` 保存
5. 进入 UOOC 课程学习页面（`https://www.uooc.net.cn/home/learn/...`）
6. 脚本会自动运行，在浏览器控制台（F12 → Console）可查看日志

## 注意事项

- 仅适用于 `uooc.net.cn` 域名下的 `/home/learn` 路径(也就是新版ui)
- 脚本会自动跳过包含"测试"或"测验"的章节(只能刷网课视频,请定时自己做测试题)
- 建议在电脑前观察第一次运行是否正常
- 记得检查tampermonkey拓展打开了开发人员模式和允许访问文件URL以及允许用户脚本
